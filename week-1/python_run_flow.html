Python 3

How print('hello world') Runs

from the editor → the interpreter → the screen
1
Write — VS Code editor
You type one line of code
In the editor you write your program and save it to a .py file. Right now it is just plain text on disk — nothing has run yet.
hello.py
1print('hello world')
2
Run — python.exe (CPython interpreter)
The interpreter reads & executes the file
Running python hello.py launches python.exe. It doesn't run your text directly — it first translates it into low-level instructions, then a virtual machine steps through them and reaches the call to print().
source text→ tokens→ AST→ bytecode→ PVM runs it
3
print() — built-in I/O function
print() turns the value into output
print() is one of Python's built-in functions. It converts its argument to a string, adds a trailing newline \n, and writes the text to standard output (sys.stdout).
sys.stdout.write('hello world\n')
4
Output — your terminal
The terminal shows the result
Standard output is wired to your terminal. The buffered text is flushed, and the characters appear on screen:
Terminal
$ python hello.py
hello world
edit → interpret → call print() → write to stdout → terminal
the same path every Python program follows, scaled down to one line
